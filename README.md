# Getting Started

Demo: https://iddog-jyrsfykstw.now.sh/

## Installing

Install dependencies
```sh
npm install
```

Run the app
```sh
npm start
```

Optionally, you can configure the API URL in the file `.env`

## Testing

```sh
npm run test
```

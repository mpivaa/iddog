# Getting Started

Install Typescript and Parcel

```sh
npm install -g typescript parcel-bundler
```

Install dependencies
```sh
npm install
```

Run the app
```sh
npm start
```

## Testing

```sh
npm run test
```

## Linter

```sh
npm run lint
```
